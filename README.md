# Superstore Sales Analysis 🛒

## Project Overview
Analyzed a real retail dataset of 9,694 sales records to find 
business insights on sales, profit, products, and customer segments.

## Tools Used
- MySQL — Data extraction and analysis
- Microsoft Excel — Pivot Tables and Charts

## Business Questions Answered
1. Which region has highest sales and profit?
2. Which are the top 5 and bottom 5 products by profit?
3. Which category has the best profit margin?
4. Which months have highest sales?
5. Which customer segment is most valuable?

## Key Findings
- West region generates highest sales (7,25,457) and profit (1,08,418)
- Canon imageCLASS 2200 Copier is the most profitable product (25,199 profit)
- Cubify 3D Printers are losing money (-8,879 loss)
- Furniture category has only 2.32% profit margin despite high sales
- November is the best sales month (3,48,834)
- Consumer segment drives maximum profit (1,32,669)
